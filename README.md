# Keziah Mariam Raison Portfolio Website

A premium, modern, and interactive personal portfolio website built using HTML, CSS, and JavaScript. This portfolio showcases skills, projects, leadership roles, contact information, and professional achievements with a strong focus on elegant UI/UX, smooth animations, dark/light mode support, and responsive design.

---

## Features

* Premium dark and light mode design
* Fully responsive layout for desktop, tablet, and mobile
* Animated preloader screen with initials and loading percentage
* Cursor glow effect for interactive experience
* Smooth scroll progress bar
* Active navigation highlighting while scrolling
* Typing text animation in hero section
* Floating technology icons around profile image
* Animated counters in About section
* Skills section with animated progress bars
* Filterable projects section
* Leadership and impact timeline cards
* Contact section with social links and contact form
* Resume download button
* Back-to-top floating button
* Elegant glassmorphism cards and hover animations

## How to Run the Project

1. Download or clone the project.
2. Open the project folder in VS Code.
3. Make sure all files like `Mine.jpg` and the resume PDF are available in the correct folder.
4. Open `index.html` in your browser.
5. For better development experience, use the Live Server extension in VS Code.
